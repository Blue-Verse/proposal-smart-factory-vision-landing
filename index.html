프로젝트 소개

스마트 팩토리 비전 검사 서비스의 시장 확대를 위한 B2B 산업 솔루션 랜딩 페이지를 HubSpot CMS Hub 기반으로 구축하여 CRM과 유기적으로 연동되는 마케팅 기반을 마련합니다.

핵심 요구사항 분석

  • 참고 사이트(overview.ai)의 모던·전문적 디자인 방향성 반영, 범위는 절반 수준의 핵심 3 메뉴로 구성
  • 핵심 메뉴 3종 구축: ① 회사/서비스 소개 ② 주요 기술/솔루션 ③ 문의하기/연락처
  • HubSpot CMS Hub 기반 콘텐츠 관리 시스템 구축 (CRM과 유기적 연동, 비개발자 운영 가능)
  • 문의 양식 → HubSpot CRM 자동 연동 (Lead 자동 생성, 워크플로우 트리거)
  • 향후 블로그 기능 확장 기반 마련 — HubSpot Blog 모듈 사전 셋업
  • PC/Mobile Web 반응형 구현, 페이지 로딩 속도 최적화
  • 내부 제공 기획/디자인 가이드를 정확히 준수하여 구현

제안 포인트

  1. 참고 사이트 기반 분석 완료 — overview.ai의 IA(Products·Solutions·Resources·Company), 히어로 + 제품 쇼케이스 + 산업 적용 사례 + 신뢰감 로고/증언 패턴, "Schedule a Demo" 중심의 전환 동선 파악 → 본 프로젝트 3 메뉴 구조에 절반 규모로 압축 반영
  2. HubSpot CMS Hub 우선 제안 — CRM/마케팅 허브와 동일 환경에 구축하여 폼 데이터·세그먼트·워크플로우가 즉시 연결, 별도 미들웨어 없이 마케팅 자동화 가능
  3. 블로그 확장 기반 사전 셋업 — Blog Listing/Post 템플릿, 카테고리/태그 분류, RSS, SEO 메타·OG 자동화까지 포함 → 추가 비용 없이 추후 블로그 콘텐츠만 추가하면 즉시 운영 가능
  4. B2B 산업 솔루션 톤앤매너 — 산업용 카메라/AI 검사·결함 탐지·엣지 추론 등 전문 용어를 신뢰감 있게 전달하는 카피·시각 구조로 구성
  5. 유사 프로젝트 수행 경험 — 50+ 페이지 B2B 포탈/대시보드, 다국어 콘텐츠 관리, 폼-CRM 연동 노하우 보유 → 20일 일정에 안정적 납품

PRD (Product Requirements Document)

프로젝트 요구사항을 정의하고 기능 범위를 명확히 합니다.

1. 프로젝트 배경 및 목적

  • 배경 — 스마트 팩토리 비전 검사 서비스의 시장 확대를 위해 새로운 랜딩 페이지가 필요. 현재 HubSpot CRM 운영 중이므로 동일 환경에 홈페이지를 구축해 마케팅 자동화·리드 관리와 자연스럽게 연결
  • 목적 — 서비스의 전문성·혁신성을 효과적으로 홍보하고 잠재 고객과의 접점 확대
  • 참고 사이트 — overview.ai의 모던·전문적 디자인 방향을 반영하되 절반 규모의 핵심 3 메뉴로 압축
  • 장기 목표 — 향후 블로그 추가 확장을 통한 콘텐츠 마케팅 기반 마련

2. 사용자 정의

사용자 유형설명주요 기능
잠재 고객 (B2B)스마트 팩토리·비전 검사 솔루션을 검토 중인 산업·제조·품질 담당자회사·서비스 정보 열람, 기술/솔루션 상세 확인, 문의 양식 제출
신규 방문자 (콜드)광고/SEO/추천을 통해 처음 유입된 사용자핵심 가치 제안 빠르게 파악, 신뢰 요소(고객사·증언) 확인

3. 기능 요구사항

3.1 회사/서비스 소개 페이지

  • [필수] 히어로 섹션 — 핵심 슬로건, 메인 비주얼, 1차 CTA(문의/데모 요청)
  • [필수] 회사 개요·미션·핵심 가치 모듈
  • [필수] 비전 검사 서비스 핵심 가치 3~4종 카드 모듈
  • [필수] 적용 산업/고객사 로고 그리드 (신뢰 요소)
  • [선택] 고객 증언/케이스 스터디 카드 영역

3.2 주요 기술/솔루션 페이지

  • [필수] 핵심 기술 특징 카드 (AI 비전, 결함 탐지, 엣지 추론 등)
  • [필수] 솔루션 다이어그램/플로우 영역 (이미지·SVG)
  • [필수] 도입 효과/지표 영역 (수치 강조 카드)
  • [선택] 데모 영상/이미지 갤러리 영역

3.3 문의하기 페이지

  • [필수] HubSpot Form 임베드 (이름·회사·이메일·연락처·문의 내용)
  • [필수] 폼 제출 → HubSpot CRM Lead 자동 생성, 담당자 이메일 알림
  • [필수] 회사 연락처·이메일·주소 정보 (지도 임베드 선택)
  • [선택] 개인정보 처리방침 동의 체크박스 (필수 동의)

3.4 블로그 확장 기반 (사전 셋업)

  • [필수] HubSpot Blog Listing 템플릿 (페이징, 카테고리/태그 필터)
  • [필수] Blog Post 상세 템플릿 (작성자, 발행일, 관련 글)
  • [필수] RSS 피드, SEO 메타/OG 태그 자동화
  • [선택] 블로그 글 발행 가이드 문서 1부 제공

3.5 공통 모듈

  • [필수] 헤더/푸터 글로벌 모듈 (메뉴 3종, 로고, 푸터 정보)
  • [필수] 반응형 (PC ≥1280px / Tablet 768~1279px / Mobile ≤767px)
  • [필수] 404 / 검색 결과 / 폼 제출 완료 페이지
  • [필수] Google Analytics 4 셋업, HubSpot Tracking Code 자동 적용

5. 기술 제약사항

  • HubSpot CMS Hub 라이선스 및 도메인 연결 권한은 클라이언트 측에서 제공
  • HubSpot CMS 라이선스 등급에 따라 일부 고급 기능(스마트 콘텐츠, A/B 테스트)은 별도
  • HubSpot 활용이 어려운 경우 WordPress / 아임웹 기반 대안 제안 — 블로그 확장성 동일하게 보장
  • 디자인은 클라이언트 내부 제공 가이드를 기준으로 구현 (디자인 신규 창작은 범위 외)
  • 다국어(영문/일문 등) 미포함 — 국문 사이트 1종 기준
